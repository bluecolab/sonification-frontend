# sonification-flask
flask site implementing sonification

How to use:
1. Read requirements.txt file
2. Run app.py with preloaded data files (Requires one csv file to be named "Default")

How to set up file:
1. If you want to have a description: Add a new column in the csv file called "description". Then add a description to the cell right below it.

For convertions from JSON to CSV:
Use https://data.page/json/csv for best results
